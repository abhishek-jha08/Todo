# 📝 JS To-Do List

A clean, simple, and responsive To‑Do List application built with HTML, CSS, and vanilla JavaScript. This project demonstrates DOM manipulation, event handling, and data persistence using the browser's localStorage.

## ✨ Features
- **Add Tasks**: Add new tasks using the input form.
- **Complete Tasks**: Click on a task to toggle completion (strikethrough styling).
- **Delete Tasks**: Remove tasks using the delete button.
- **Local Storage**: Tasks persist across page reloads and browser sessions.
- **Responsive Design**: Mobile-friendly layout using flexible CSS.

## 🚀 Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Custom styling and responsive layout (Flexbox/Grid).
- **JavaScript (ES6+)**: Core logic for task management and localStorage.
- **FontAwesome**: Icons for UI elements (optional).

## 💡 Usage
1. Open `index.html` in your browser.
2. Type a task in the input field and press Enter or click the Add button.
3. Click a task to toggle its completion state.
4. Click the delete icon to remove a task.
